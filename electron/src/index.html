General

Cuanta libertad e iniciativa tiene el agente para realizar actividades.
El agente solo actua cuando le pides una tarea operativa. Un "Hola" nunca dispara acciones.
Decide cuando el agente pide tu aprobacion antes de ejecutar una accion.
Pantalla donde el agente toma screenshots y ejecuta acciones. Cada monitor tiene su propia resolucion.

Modelo IA

AI Studio usa API key (gratis con límites). Vertex AI usa credenciales GCP y consume los créditos de Google Cloud ($300 Free Trial).
—
Usa "global" para Gemini 3.x preview. Las regiones específicas solo tienen Gemini 2.5.
Si se deja vacío, usa Application Default Credentials (gcloud auth application-default login)
Elige los modelos que el agente utilizará cuando deba generar imágenes, video o música.
Elige qué proveedor y modelo usarán los sub-agentes para cada tipo de tarea. Si no se configura, se usa el modelo por defecto.
Sub-agente dedicado a la interacción de UI (clicks, typing, scrolling). El agente principal delega aquí cuando no usa MCPControl. Soporta varios proveedores: elige proveedor y modelo de computer use.

Voz y Personaje

Elige la voz que usará Google Gemini TTS en todas las respuestas.
Usa las voces instaladas en tu Windows/navegador. Sin red, instantáneo.
-
Google TTS usa la misma API key del provider Google/Gemini.
-
En modo avatar, la ventana de chat se oculta y aparece un personaje flotante en el escritorio.

Equipos Multi-Agente

Los equipos permiten combinar múltiples sub-agentes con roles especializados para tareas complejas. Proceso secuencial: las tareas se ejecutan en orden. Jerárquico: un manager delega a los agentes.
